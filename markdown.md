# Сервис бронирования переговорных комнат

Веб-сервис для автоматизации бронирования комнат в коворкинге на FastAPI.

## Возможности

- Аутентификация по JWT (роли: сотрудник, администратор).
- Просмотр списка комнат и доступных слотов на дату.
- Бронирование и отмена своих броней.
- Администратор может отменить любую бронь.
- Данные хранятся в PostgreSQL через SQLAlchemy.
